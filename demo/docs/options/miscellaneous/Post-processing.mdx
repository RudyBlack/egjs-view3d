Post-processing applies various effects to rendered scene.

View3D offers post-processing on THREE.JS, including SSR, DoF, Bloom, and SAO.

```js
import View3D, {SSR, DoF, Bloom} from "@egjs/view3d";

const view3D = new View3D(...);

view3D.loadEffects(
    new SSR(),
    new DoF(),
    new Bloom({ radius: 0.5, threshold: 0.3, strength: 1 }),
);

```
:::caution
SAO and SSR can not be used together.
:::

Also, you can use customize THREE.JS post-processing or [Postprocessing](https://github.com/pmndrs/postprocessing) of library.

```js
import View3D from "@egjs/view3d";

// It's EffectComposer of built-in three.js
import { EffectComposer } from "three/examples/jsm/postprocessing/EffectComposer";
// Or you can use EffectComposer of postprocessing library
import { EffectComposer } from "postprocessing";

const view3D = new View3D(...);

view3D.setCustomEffect((param) => {
      const { camera, renderer, canvasSize, scene, model } = param;
      const effectComposer = new EffectComposer(renderer);

      ...

      // You have to return EffectComposer
      return effectComposer;
});
```
